# The Turing Circle Website

This project is a Next.js website for The Turing Circle, the Math and Computing Club of MIT Manipal.

## Prerequisites

- **Node.js**: You need to have Node.js installed on your computer. Download it from [nodejs.org](https://nodejs.org/).

## Setup & Run

1.  Open this folder in your terminal (Command Prompt or PowerShell).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 14**: Framework
- **Tailwind CSS**: Styling
- **GSAP & ScrollTrigger**: Animations
- **Lenis**: Smooth Scrolling
- **Framer Motion**: Micro-interactions

## Project Structure

- `src/app`: Pages and global styles.
- `src/components`: Reusable UI components.
- `public/assets`: Images and static assets.
